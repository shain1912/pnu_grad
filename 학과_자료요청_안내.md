# 대학원 학과 자료 요청

부산대학교 학·석사 연계과정 신청 사이트의 **"어떤 학과가 있는가"** 화면(계열 → 학과 → 세부전공 선택, 학과 클릭 시 상세 팝업)에 들어갈 자료입니다.
**현재 학과 데이터가 없는 상태로 간주하고, 학과 목록부터 처음 전부 작성**해 주시면 됩니다.

## ★ 작성 단위: **세부전공 한 개당 한 줄**
- 세부전공이 여러 개인 학과는 **전공 수만큼 행을 나눠** 적어 주세요. (같은 학과면 `계열`·`학과명`은 반복 기재)
- **세부전공이 없는(학과 단위) 학과**는 `세부전공` 칸을 비우고 **한 줄**만 적습니다.
- **학과 단위 정보**(해시태그·한 줄 설명·홈페이지·정원·대표 이미지·BK21 항목)는 **그 학과의 첫 줄에만** 적어 주세요. (나머지 전공 행은 비워두면 됩니다 — 저희가 학과별로 취합합니다.)

## 열 설명 (동봉 `학과_자료요청_템플릿.csv`)

| 열 | 내용 | 가이드 | 단위 |
|---|---|---|---|
| **계열** | 학과가 속한 계열 | `인문·사회` / `자연과학` / `공학` / `예술` / `체육` / `의학` 중 1 | 모든 줄 |
| **학과명** | 일반대학원 학과(부)명 | 모집요강상 정식 명칭 | 모든 줄 |
| **세부전공** | 세부전공 1개 | 없으면 공란 | **줄마다 1개** |
| **모집여부** | 그 전공 모집 여부 | `O`(모집) / `X`(미모집) | 전공별 |
| **해시태그** | 학과 키워드 3~5개 | 예: `#반도체 #AI #소재` | 학과 첫 줄만 |
| **한 줄 설명** | 학과를 한 문장으로 | 60자 내외 | 학과 첫 줄만 |
| **홈페이지 URL** | 학과 공식 홈페이지 | `https://` 포함 | 학과 첫 줄만 |
| **정원(명)** | 연계과정 모집 인원/정원 | 숫자 | 학과 첫 줄만 |
| **대표 이미지** | 학과 대표 사진 | 1:1, 1600px↑ · 파일명만 | 학과 첫 줄만 |
| **BK21 참여** | BK21 사업단 참여 여부 | `Y` / `N` | 학과 첫 줄만 |
| **BK21 사업단명** | (참여 학과만) 정식 사업단명 | | 학과 첫 줄만 |
| **BK21 홈페이지** | (참여 학과만) 사업단 홈페이지 | `https://` 포함 | 학과 첫 줄만 |

## 작성 예시 (템플릿 상단 `[예시]` 행 참고)
```
계열  학과명     세부전공        모집  해시태그        설명     홈페이지  정원 이미지   BK21 ...
공학  기계공학부  에너지시스템      O    #기계 #로봇...   (설명)   (URL)    30  (파일) Y  (사업단)
공학  기계공학부  기계시스템설계    O    (빈칸 — 학과 첫 줄에 이미 적음)
공학  기계공학부  지능형제조시스템   X    (빈칸)
인문·사회 철학과   (공란=세부전공 없음) O  #철학...        (설명)   (URL)     8 (파일) N
```
→ 기계공학부는 전공 3개라 **3줄**, 학과 정보는 첫 줄에만.
대표 이미지 파일은 시트엔 파일명만, 이미지 파일은 별도(압축 등)로 함께 전달해 주세요.
